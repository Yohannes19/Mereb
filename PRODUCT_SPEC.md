# Product Spec

## Mission

Help Ethiopian creators, freelancers, and small businesses prove their reputation online.

## Product Summary

A user creates a public profile, adds finished work as proof items, collects testimonials and ratings, and shares one link with potential clients.

## Target Users

- Creators
- Photographers
- Designers
- Video editors
- Makeup artists
- Event planners
- Small agencies
- Small businesses that want a more trusted online presence

## Core Problem

Most people sell through informal channels:

- DMs
- Telegram
- Instagram
- WhatsApp
- word of mouth

They can say they have done good work, but they often cannot show structured proof in one trusted place.

## Value Proposition

ProofPage turns past work into a public reputation asset.

## Differentiation

- Public trust profile instead of a generic portfolio
- Ratings and testimonials on the same page
- Proof-item format built around real outcomes and links
- Works for creators and small businesses, not just one category

## V1 Functional Requirements

- Account registration and login
- Session-based dashboard access
- API key authentication
- Profile editing
- Add proof item
- Add rating / testimonial
- Public proof page
- Proof and rating APIs

## V1 Non-Functional Requirements

- Mobile-friendly public profile
- Easy to share in DMs
- Clear, trust-building UI
- Simple deployment later

## Out of Scope for V1

- Messaging inbox
- Job marketplace
- Payments
- Contracts
- Verification workflow with human review
- Search and discovery marketplace

## Success Metrics

- User can create a profile in under 5 minutes
- User can add first proof item in under 3 minutes
- Public page looks credible enough to share immediately
- Users return to add more projects and testimonials

## Product Principle

Do not position this as a social network.

Position it as:

The proof link for Ethiopian work and reputation.
